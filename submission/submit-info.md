## 作品名称

VoxCanvas Lab - AI 语音绘图工具

## 选择题目

第四批次题目二：AI 语音绘图工具

## 作品简介

VoxCanvas Lab 是一款纯语音控制的浏览器绘图工具。用户启动麦克风后，通过自然语言说出绘图意图，系统在本地完成语音识别结果解析、复杂指令拆解、Canvas 绘制与语音反馈。作品重点解决语音指令理解、容错、复杂命令拆解、绘图响应延迟和运营成本控制问题。

## 技术栈

- HTML / CSS / JavaScript
- Web Speech API
- Canvas 2D
- 本地规则语义解析器
